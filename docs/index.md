# Bienvenida

Bienvenidos a la unidad de Programación. Hoy en día, la tecnología rodea casi todo lo que hacemos, desde el móvil hasta el funcionamiento de un semáforo o de un coche. Pero, ¿cómo saben estas máquinas lo que tienen que hacer? La respuesta está en la programación.

## Introducción

Vivimos rodeados de tecnología: móviles, videojuegos, redes sociales, robots… pero, ¿te has preguntado alguna vez cómo “saben” las máquinas lo que tienen que hacer?

La respuesta está en los **algoritmos**. Un algoritmo es simplemente una serie de pasos ordenados para resolver un problema o realizar una tarea. Por ejemplo, seguir una receta de cocina o las instrucciones para montar un mueble también son algoritmos.

Para poder entenderlos mejor y comunicarlos de forma clara, utilizamos dos herramientas muy importantes:

* **Diagramas de flujo** 🧩: representan los pasos de un algoritmo mediante dibujos y símbolos, como si fuera un mapa visual del proceso.
* **Pseudocódigo** 💻: es una forma de escribir algoritmos usando un lenguaje parecido al código, pero más sencillo y fácil de entender.

En este curso aprenderás a pensar como un programador: descomponer problemas, organizar ideas y crear soluciones paso a paso. No solo te servirá para programar, sino también para resolver problemas en tu vida diaria de forma más lógica y eficiente.
